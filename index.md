# User Dashboard Pro

User panel with admin panel


## Features

### User panel
- Simple Login & Register
- Detect IP
- User active time
- Trade coin with other user
- Redeem coin
- IAP status

### Admin panel
- Create redeem code
- Send coin to user
- Redeem code
